# サービス名
学習記録一覧アプリ
# サービスの説明
プログラミングを習得するには1000時間を
必要とするらしくそれまでの学習時間を記録するアプリを作成しまいた。
このアプリは学習内容と学習時間を記録することができます。
# サービスの使用方法
1.学習内容と学習時間を記載します。
2.登録ボタンをクリックします。
登録ボタンの上部に入力した値が表示されます。
データはsupabaseに送られます。
削除ボタンをクリックすることで削除することができます。
# 環境設定の方法
```
$ git clone https://github.com/kitamura567/Chapter1-2.git
$ cd Chapter1-2
```
npmのインストール
```
$ npm install
```
.envで環境変数を設定してください
```
VITE_SUPABASE_URL="あなたのURL"
VITE_SUPABASE_ANON_KEY="あなたのKEY"
```
# テストの実行
```
$ npm test
```

# 起動方法
```
$ npm run dev
```
